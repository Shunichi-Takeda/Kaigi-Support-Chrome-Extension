# Kaigi-Support-Chrome-Extension

Googleカレンダーの予定作成画面に「AIアジェンダ生成パネル」を統合するChrome拡張機能です。
Gemini Nano（ブラウザ内蔵AI）を活用し、会議の目的に応じた最適なアジェンダ・ゴール・役割分担を自動生成します。

## 特徴

### 5つの会議モード

| モード | 入力項目 | AI生成 | 役割分担 | ゴール | 3つの約束 |
|---|---|---|---|---|---|
| **アイデア出し** | 場所・やりたいこと・ゴール・資料URL・時間 | ✅ アジェンダ＋ゴール＋タイトル | ✅ | ✅ 完了形式×3観点 | ✅ |
| **意思決定** | 場所・やりたいこと・ゴール・資料URL・時間 | ✅ アジェンダ＋ゴール＋タイトル | ✅ | ✅ 完了形式×3観点 | ✅ |
| **情報共有・調整** | 場所・やりたいこと・ゴール・資料URL・時間 | ✅ アジェンダ＋ゴール＋タイトル | ✅ | ✅ 完了形式×3観点 | ✅ |
| **1on1** | 場所・時間 | ❌ 固定テンプレート | ❌ | ❌ | ✅ |
| **自由記入** | 場所・やりたいこと | ✅ アジェンダのみ | ❌ | ❌ | ❌ |

### 各モードの詳細

- **アイデア出し / 意思決定 / 情報共有・調整**
  - 会議種別ごとに最適化された**アジェンダ構成ロジック**（3ステップ）と**ゴール策定観点**（3観点）をAIプロンプトに組み込み
  - **役割分担**: 3つの名前付き役割 ＋「その他の参加者」のスタンスを自動提案
  - **ゴール**: 「〜ている状態」の完了形式で、結論・行動・納得感の3観点から出力
  - **タイトル**: AIが35文字以内で自動生成
  - **終了時刻**: 指定した会議時間に基づき自動更新

- **1on1モード**
  - 1on1を形骸化させないための**「7つの目的カテゴリー」**をアジェンダに自動挿入
  - やりたいこと・ゴール・資料URL欄は入力不可（固定テンプレート使用）
  - タイトルは「場所 + やりたいこと」から設定

- **自由記入モード**
  - 「場所」と「やりたいこと」の2項目のみからAIでアジェンダを生成
  - 時間配分・ゴール・資料URL・役割分担・3つの約束はすべて省略
  - シンプルな会議やメモ的利用に最適

### その他の機能

- **簡易ポップアップ対応**: 予定作成のポップアップにパネルを自動表示（横幅1.5倍拡張）
- **「価値ある時間」にするための3つの約束**: 通常会議のアジェンダ末尾にマインドセットを自動付与
- **プレーンテキスト出力**: マークダウン記法を使わず、Googleカレンダーの説明欄に最適化
- **AI出力サニタイズ**: 文字化け防止のため特殊文字を自動除去
- **ダークモード対応**: システム設定に連動

## 使い方

1. Googleカレンダーで予定作成のポップアップを開きます
2. パネル上部で「会議の種類」を選択します
3. 必要な情報を入力し、Geminiボタン（✦）をクリックします
4. アジェンダ・タイトル・終了時刻が自動で反映されます

## 導入方法

### Chrome Flagsの設定（AI機能の有効化）

1. `chrome://flags/#optimization-guide-on-device-model` → **Enabled**
2. `chrome://flags/#prompt-api-for-gemini-nano` → **Enabled**
3. Chromeを再起動

### 拡張機能のインストール

1. 本リポジトリをダウンロードして解凍します
2. `chrome://extensions/` を開きます
3. 「デベロッパーモード」をオンにします
4. 「パッケージ化されていない拡張機能を読み込む」で解凍したフォルダを指定します

## 技術仕様

- **Manifest V3** 対応
- **Gemini Nano** (LanguageModel API) によるブラウザ内蔵AI生成
- `MutationObserver` による動的なDOM検知
- Google Calendarのフレームワークに適合したイベントシミュレーション（`focus` → `input` → `change` → `blur`）
- AI出力のサニタイズ（絵文字・特殊文字・制御文字の除去）
- ダークモード対応CSSカスタムプロパティ

## ファイル構成

| ファイル | 役割 |
|---|---|
| `manifest.json` | 拡張機能の定義（Manifest V3） |
| `content.js` | メインロジック（UI生成・AI連携・DOM操作） |
| `background.js` | Service Worker（AI API管理・メッセージング） |
| `styles.css` | パネルのスタイル定義 |
