# Kaigi-Support-Chrome-Extension

Googleカレンダーの予定作成画面に「アジェンダ作成支援パネル」を統合するChrome拡張機能です。
通常の会議だけでなく、**「1on1」の質を向上させるための専用モード**や、会議を活性化させるための**「3つの約束」**の自動挿入機能を備えています。

## 特徴

- **3つの会議種別 + 1on1サポート**:
  - 「アイデア出し」「意思決定」「情報共有・調整」の各モードに最適化された役割分担を提案。
  - **1on1モード**: 1on1を形骸化させないための「7つの目的カテゴリー」をアジェンダに自動挿入。
- **「価値ある時間」にするための3つの約束**:
  通常の会議アジェンダの末尾に、チームの熱量を高めるためのマインドセットを自動付与。
- **動的なUI**: 選択した会議の種類に応じて入力項目が最適化されます。
- **ワンクリック反映**: 生成したアジェンダをGoogleカレンダーの「説明」欄に即座に反映。

## 導入方法（配布・インストール手順）

複数人で利用する場合や、非開発者の方が導入する場合は以下の手順で行ってください。

### 1. 拡張機能のダウンロード
1. 本リポジトリの [Releases](../../releases) ページにアクセスします。
2. 最新バージョンの「Assets」から **Source code (zip)** をダウンロードします。
3. ダウンロードした zip ファイルを、PC内の任意の場所（ドキュメントフォルダなど、削除しない場所）に解凍します。

### 2. Chromeへのインストール
1. Google Chrome を開き、アドレスバーに `chrome://extensions/` を入力して移動します。
2. 右上の **「デベロッパーモード」** をオンにします。
3. 左上の **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
4. 先ほど解凍したフォルダを選択します。

### 3. アップデート方法（最新版への更新）
新しいバージョンがリリースされた場合は、以下の手順で更新できます。
1. 新しいバージョンの zip ファイルをダウンロードし、解凍します。
2. **既存のフォルダの内容を、新しい内容で上書きします。**
3. `chrome://extensions/` 画面を開き、本拡張機能のカードにある **「更新（丸い矢印アイコン）」** をクリックします。

## 技術仕様

1. Googleカレンダーで予定作成、または編集画面を開きます。
2. 「説明を追加」の近くに表示されるパネルで「種類」を選択します。
3. 必要に応じて情報を入力し、「文言生成」→「反映」の順にクリックします。

## トラブルシューティング

パネルが表示されない場合は、以下の点を確認してください：
- **ページの再読み込み**: 拡張機能を更新した後は、Googleカレンダーのタブを更新してください。
- **詳細画面の確認**: 簡易ポップアップで表示されない場合、「その他のオプション」をクリックして詳細編集画面を開いてみてください。
- **コンソールの確認**: `F12` キーでデベロッパーツールを開き、`Console` タブに `[KSCE] Content script loaded.` と表示されているか確認してください。
