# Kaigi-Support-Chrome-Extension

Googleカレンダーの予定作成画面に「アジェンダ作成支援パネル」を統合するChrome拡張機能です。
通常の会議だけでなく、**「1on1」の質を向上させるための専用モード**や、会議を活性化させるための**「3つの約束」**の自動挿入機能を備えています。

## 特徴

- **3つの会議種別 + 1on1サポート**:
  - 「アイデア出し」「意思決定」「情報共有・調整」の各モードに最適化された役割分担を提案。
  - **1on1モード**: 1on1を形骸化させないための「7つの目的カテゴリー」をアジェンダに自動挿入。
- **1on1の7つの目的カテゴリー**:
  1. 信頼関係の構築（プライベート・相互理解）
  2. 心身のコンディション確認（メンタル・健康）
  3. 業務課題の解決・支援（障害の除去）
  4. 戦略・目標のすり合わせ（アライメント）
  5. フィードバック（評価と改善）
  6. 能力開発・育成（スキルアップ）
  7. キャリア・ビジョン（中長期の展望）
- **「価値ある時間」にするための3つの約束**:
  通常の会議アジェンダの末尾に、チームの熱量を高めるためのマインドセットを自動付与。
- **動的なUI**: 選択した会議の種類に応じて入力項目が最適化されます（1on1選択時は不要な項目を非表示化）。
- **ワンクリック反映**: 生成したアジェンダをGoogleカレンダーの「説明」欄に即座に反映。

## 導入の目的

多くの会議や1on1が「何を話すべきか不明確」「記録が残らない」「当事者意識が低い」といった課題を抱えています。本ツールは、カレンダー入力という業務の起点において適切なフレームワークを提示することで、準備の負荷を下げつつ議論の質を最大化することを目的としています。

## インストール方法（デベロッパーモード）

1. このリポジトリをローカルにダウンロードまたはクローンします。
2. Google Chromeを開き、アドレスバーに `chrome://extensions/` を入力します。
3. 右上の「デベロッパーモード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、本プロジェクトのフォルダを選択します。

## 使い方

1. Googleカレンダーで予定作成、または編集画面を開きます。
2. 注入されたパネルで「種類」を選択します。
3. 必要に応じて「概要」などを入力し、「文言生成」をクリックします。
4. プレビューを確認し、「反映」をクリックしてカレンダーの説明欄に流し込みます。
