# Kaigi-Support-Chrome-Extension

Googleカレンダーの予定作成画面に「アジェンダ作成支援パネル」を統合するChrome拡張機能です。
通常の会議だけでなく、**「1on1」の質を向上させるための専用モード**や、会議を活性化させるための**「3つの約束」**の自動挿入機能を備えています。

## 特徴

- **3つの会議種別 + 1on1サポート**:
  - 「アイデア出し」「意思決定」「情報共有・調整」の各モードに最適化された役割分担を提案。
  - **1on1モード**: 1on1を形骸化させないための「7つの目的カテゴリー」をアジェンダに自動挿入。
- **クイック作成画面にも対応**:
  - 予定の簡易ポップアップ時にも、右側に支援パネルを自動で表示します。
- **「価値ある時間」にするための3つの約束**:
  通常の会議アジェンダの末尾に、チームの熱量を高めるためのマインドセットを自動付与。
- **動的なUI**: 選択した会議の種類に応じて入力項目が最適化されます。
- **ワンクリック反映**: 生成したアジェンダをGoogleカレンダーの「説明」欄に即座に反映。

## 使い方

1. Googleカレンダーで予定作成、または編集画面を開きます。
2. **クイック作成（ポップアップ）:** 画面の右側にパネルが表示されます。
3. **詳細編集画面:** 「説明を追加」セクションのすぐ上にパネルが埋め込まれます。
4. パネルで「種類」を選択し、情報を入力後に「文言生成」→「反映」の順にクリックします。

## 導入方法

1. 本リポジトリをダウンロードして解凍します。
2. Google Chromeのアドレスバーに `chrome://extensions/` を入力します。
3. 右上の「デベロッパーモード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」を選択し、解凍したフォルダを指定します。

## 技術仕様

- **Manifest V3** 対応。
- `MutationObserver` による動的なDOM検知。
- Googleカレンダーのスタイルに馴染むモダンなUIデザイン。
