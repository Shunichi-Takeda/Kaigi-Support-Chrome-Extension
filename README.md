# Kaigi-Support-Chrome-Extension

Googleカレンダーの予定作成画面に「アジェンダ作成支援パネル」を統合するChrome拡張機能です。会議の目的に合わせたアジェンダ構成と役割分担を自動で提案し、会議の質と生産性を向上させます。

## 特徴

- **3つの会議種別サポート**: 「アイデア出し」「意思決定」「情報共有・調整」の3つのモードから選択可能。
- **動的な役割提案**: 会議の種類に応じて、進行役や記録係などの代表的な役割を自動的に挿入。
- **AI連携準備済み**: 将来的なAIによる文言生成を見据えた非同期処理構造。
- **ワンクリック反映**: 生成したアジェンダをワンクリックでカレンダーの「説明」欄に挿入（上書き確認機能付き）。
- **シームレスなUI**: Googleカレンダーのモダンなデザインに馴染む直感的なインターフェース。

## インストール方法（デベロッパーモード）

1. このリポジトリをローカルにダウンロードまたはクローンします。
2. Google Chromeを開き、アドレスバーに `chrome://extensions/` を入力して移動します。
3. 右上の「デベロッパーモード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリックします。
5. ダウンロードしたプロジェクトのフォルダを選択します。

## 使い方

1. Googleカレンダー（https://calendar.google.com/）を開きます。
2. 予定を作成、または既存の予定をクリックして編集画面を開きます。
3. 「説明」フィールドの付近に表示される「会議アジェンダ作成支援」パネルを確認します。
4. 会議の「種類」「概要」「ゴール」などを入力します。
5. 「文言生成」ボタンをクリックすると、入力内容に基づいたアジェンダのプレビューが表示されます。
6. 「反映」ボタンをクリックし、確認ダイアログで「OK」を押すと、カレンダーの「説明」欄に内容が自動挿入されます。

## 会議種別ごとの役割例

本ツールでは、会議の目的に応じて以下の役割を意識することを推奨しています：

- **アイデア出し**: 進行（発言を促す）、記録（可視化する）、時間（テンポを作る）
- **意思決定**: 決定者（最終決断を下す）、提案者（判断材料を提示）、異論役（リスクを指摘）
- **情報共有・調整**: 報告者（現状を伝える）、調整者（利害を整える）、アクション確認（タスク復唱）
